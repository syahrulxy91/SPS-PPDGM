# Background Assets

Sila simpan fail imej latar belakang (contoh: `login-bg.jpg` atau `login-bg.png`) di dalam folder ini untuk membolehkan ia dipaparkan pada halaman log masuk (Landing Page).

## Kegunaan:
- Latar belakang log masuk sistem e-Laporan SPS.
- Boleh dihantar secara dinamik mengikut keperluan rasmi.
